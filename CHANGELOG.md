## Upcoming Releases

### [1.0.0] - Planned (Week 4)
**MVP Release** - Basic functional prototype

#### Planned Features
- Camera capture and image upload
- Azure Computer Vision object detection
- Rule-based verb suggestions
- Sentence construction with templates
- Web Speech API text-to-speech
- User authentication (JWT)
- Feedback collection (thumbs up/down)
- PostgreSQL database with migrations
- Responsive UI with Tailwind CSS
- WCAG AAA accessibility compliance

#### Technical Stack
- Backend: Python 3.11 + FastAPI + SQLAlchemy
- Frontend: Vue 3 + Vite + Tailwind CSS + Pinia
- Database: PostgreSQL 14+
- Cloud: Azure (Computer Vision, App Service, PostgreSQL, Blob Storage)

#### Success Criteria
- [ ] Object detection working with 90%+ accuracy
- [ ] Sentence construction in <30 seconds
- [ ] TTS works on major browsers
- [ ] 80%+ test coverage
- [ ] Deployed to Azure staging environment
